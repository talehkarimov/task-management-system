# Task Management – Business Rules

## Core Concepts
- Organization owns projects
- Project groups tasks
- Task represents a unit of work

## Roles
- Admin: manages organization and projects
- Project Manager: manages tasks
- Member: executes tasks

## Task Lifecycle
A task follows this lifecycle:
- ToDo
- InProgress
- Blocked
- Done

## Rules
- A task belongs to exactly one project
- A task can have only one assignee
- Completed tasks cannot be modified
- All important actions are audited

## Key Scenarios
- Task creation
- Task assignment
- Task execution
- Task blocking
- Task completion
